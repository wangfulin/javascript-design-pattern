# 使用 javascript 实现23 种设计模式

设计模式的思想对于我们以后构建复杂的应用，合理规划好类之间的关系，提高鲁棒性，降低耦合性非常有帮助。我希望能够几个方面来理解和分析设计模式：

1. 核心概念
2. UML图
3. 经典应用场景
4. 代码实例
